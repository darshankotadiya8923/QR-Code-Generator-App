# 📡 QR Code Generator App

An Android application that generates QR codes instantly from text, URLs, contact details, WiFi credentials, and more.

Built using Kotlin and ZXing library.

---

## 🚀 Features

- Generate QR code from:
  - Text
  - Website URL
  - Contact Information
  - Email
  - Phone Number
  - WiFi Credentials
- Instant QR preview
- Simple and clean UI
- Works offline
- Lightweight and fast

---

## 🛠 Technologies Used

- Kotlin
- Android Studio
- XML Layout Design
- ZXing (QR Code Library)

---

## 📦 Library Used

```
implementation("com.journeyapps:zxing-android-embedded:4.3.0")
implementation("com.google.zxing:core:3.5.1")
```

---

## ▶️ How to Run the Project

1. Clone the repository:
   ```
   git clone https://github.com/YOUR_USERNAME/QR-Code-Generator-App.git
   ```
2. Open in Android Studio
3. Sync Gradle
4. Run on emulator or physical device
